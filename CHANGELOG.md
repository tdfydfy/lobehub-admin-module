# Changelog

本文件记录 `lobehub-admin-module` 的主要版本变化。旧版 README 中混杂的更新记录已迁移到这里，并按日期重写整理。

## 2026-04-03

### CRM 客户小结同步

- 新增轻量 CRM 小结同步链路，将对话中的阶段性客户小结结构化写入 `crm.customer_profiles`
- 小结结构升级到 `crm_customer_summary.v2`，补齐 `gender`、`age`、`familyStructure`、`residenceArea`、`desiredLayout`、`targetUnitPrice`、`targetTotalPrice`、`firstVisitTime` 等字段
- 意向等级统一收口为 `A / B / C / D / null`
- 新增称呼兜底的性别推断，避免“张先生但性别为空”这类落库问题
- 客户小结正文上限调整为 300 汉字，跟进建议继续只用于当前对话展示，不入库
- 消息正文中的“存档状态”会随同步结果回写，便于用户确认是否已保存

### 日报与自由盘点模型配置收口

- 日报与自由盘点统一走服务端默认模型配置，不再依赖项目内单独覆盖 provider / model name
- 自由盘点兜底提示不再写死具体厂商，统一改为“默认模型”
- 修复日报提示词设置页标题乱码，统一显示为“系统提示词”
- 收敛自由盘点请求体与提示词体积，降低 `fetch failed` 风险

### 管理端顶部与生产同步

- 恢复系统管理员头部常驻项目卡与项目切换入口
- 移除首页重复的项目列表区块，减少同屏冗余
- `ali-2c2g` 已同步到当前版本

## 2026-04-02

- 新建项目时自动生成默认项目文档模板包
- 项目文档页新增“补齐默认模板”操作，只补缺失模板，不覆盖已有内容
- 默认入口文档统一为项目知识地图与项目主卡

## 2026-04-01

- 统一知识插件从“按项目一个插件”收口为“按账号唯一项目归属的统一插件”
- 新增统一公开知识查询入口 `/public/knowledge/:userId/:signature/query`
- 引入单账号单项目绑定约束，普通账号不再允许绑定多个项目
- 登录上下文补充 `activeProjectId`、`activeProjectName`、`activeProjectRole`、`bindingStatus`

## 2026-03-31

- 新增 `project_documents` 与 `global_documents`，形成项目知识与全局知识双层文档体系
- 管理端补齐项目文档、全局知识文档与统一知识插件同步链路
- 新增 `project_topic_daily_facts` 与 `project_daily_overview_view`
- 组合看板、单项目概览与移动端首页开始统一基于经营事实层输出

## 2026-03-27

- 上线“自由盘点”能力，支持项目级客户分析会话、提示词输入与后台任务执行
- 新增客户分析会话表与任务表，任务提交后改为异步执行并支持恢复
- 前端补齐项目详情页与移动端的自由盘点入口

## 2026-03-24

- 调整日报意向口径，明确 `A + B` 为中高意向，`C + D` 为中低意向
- 日报不再重复发明意向标准，优先复用对话侧已有客户判断
- 重排移动端日报详情页，强化摘要、重点客户、管理动作与返回路径

## 2026-03-21

- 上线项目经营日报能力，包括设置、手动生成、自动调度、列表与详情
- 日报正文默认接入火山引擎模型，并保留调用失败时的规则兜底摘要
- 新增移动端管理工作台，逐步对齐桌面端的经营概览能力

## 2026-03-20

- 修复 Topic 详情查询与可见消息统计口径
- Topic 详情页不再混入大量无效的中间 assistant / system 占位消息
- 成员管理页支持查看项目助手详情，包括提示词、模型配置与技能列表

## 2026-03-19

- 批量配置与刷新改为“API 快速返回 + 服务端后台执行”模式
- 为同一项目增加运行中任务并发保护，避免重复创建任务
- 批量任务在创建时固化成员快照，不再执行时重新扫描成员列表
- 明确每个 `(project_id, user_id)` 只允许 1 个官方助手和 1 个官方会话
- 新增增量 SQL：`003_fix_provision_skip_requires_session.sql`、`004_repair_project_managed_mappings.sql`

## 2026-03-18

- 修复模板助手归属校验问题
- 同步成员访问与部署文档，补齐生产环境修复说明

## 2026-03-17

- 首次提交独立外挂管理模块
- 初步落下项目管理、成员管理、模板助手配置、部署说明与生产环境示例文件
